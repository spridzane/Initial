12. exercise: Networking

  1. Review presentation Networking

      

  2. Investigate jtm.activity12 package
  3. Implement class ChatServer to pass unit tests

  1. Server could be checked using telnet program, e.g. following:

    telnet 127.0.1.1 9999

  2. To check if server is listening on necessary port, run command:

    netstat -antp|grep 9999

  3. If necessary, server can be killed with command:

    kill -9 pid
    wher pid is process id from the previous command
